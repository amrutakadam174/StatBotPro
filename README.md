# 🚀 StatBot Pro – Autonomous CSV Data Analyst

StatBot Pro is an AI-powered data analysis application that allows users to upload CSV datasets, ask questions in natural language, and receive instant statistical insights.

It acts as a smart data analyst by interpreting user queries and performing operations using Python and Pandas.

---

## 📅 Week 1 – Basic Data Analysis (Completed)

In Week 1, the core functionality of the system has been implemented, focusing on data upload, inspection, and basic query handling.

### ✅ Features Implemented

- 📁 Upload CSV datasets  
- 📊 Display dataset preview  
- 📌 Show dataset information (rows, columns, column names)  
- 💬 Accept natural language queries  
- 🧠 Perform basic statistical analysis  

---

## 🔍 Supported Queries

The system currently supports the following types of queries:

### 📊 Data Inspection
- How many rows?  
- How many columns?  

### 📈 Statistical Analysis
- Mean / Average  
  - Example: *What is the mean of Sales?*  
- Maximum  
  - Example: *Max Profit*  
- Minimum  
  - Example: *Min Quantity*  

### 💡 Natural Language Support
- What is the mean value of 'Sales'?  
- Show the maximum profit  
- Give me the minimum value of Quantity  

---

## 🏗️ Tech Stack

- **Frontend:** Streamlit  
- **Backend:** Python  
- **Data Processing:** Pandas  
- **Version Control:** Git & GitHub  

---

## ⚙️ How It Works

1. Upload a CSV file  
2. View dataset preview  
3. Enter a query in natural language  
4. System processes the query  
5. Displays the result using Pandas
